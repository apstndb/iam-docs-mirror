Some Policy Intelligence tools, like [Policy Analyzer for IAM policies](https://docs.cloud.google.com/policy-intelligence/docs/policy-analyzer-overview) and [Policy Analyzer for organization policies](https://docs.cloud.google.com/policy-intelligence/docs/analyze-organization-policies) , are part of the Cloud Asset API. You can view the reference documentation for these tools' API methods in the Cloud Asset Inventory documentation:

  - [`analyzeIamPolicy`](https://docs.cloud.google.com/asset-inventory/docs/reference/rest/v1/TopLevel/analyzeIamPolicy)
  - [`analyzeIamPolicyLongrunning`](https://docs.cloud.google.com/asset-inventory/docs/reference/rest/v1/TopLevel/analyzeIamPolicyLongrunning)
  - [`analyzeOrgPolicies`](https://docs.cloud.google.com/asset-inventory/docs/reference/rest/v1/TopLevel/analyzeOrgPolicies)
  - [`analyzeOrgPolicyGovernedContainers`](https://docs.cloud.google.com/asset-inventory/docs/reference/rest/v1/TopLevel/analyzeOrgPolicyGovernedContainers)
  - [`analyzeOrgPolicyGovernedAssets`](https://docs.cloud.google.com/asset-inventory/docs/reference/rest/v1/TopLevel/analyzeOrgPolicyGovernedAssets)
  - [`savedQueries`](https://docs.cloud.google.com/asset-inventory/docs/reference/rest/v1/savedQueries)

To view all of the methods in the Cloud Asset API, see [Cloud Asset API](https://docs.cloud.google.com/asset-inventory/docs/reference/rest) .

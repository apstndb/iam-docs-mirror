Before setting up Identity and Access Management (IAM) for your Google Cloud organization, you should understand how IAM fits in to your organization's security architecture as a whole.

To learn about Google's core security principles and find links to additional resources, see [Google Cloud Well-Architected Framework: Security, privacy, and compliance](https://docs.cloud.google.com/architecture/framework/security) in the Cloud Architecture Center. This information can help you architect your Google Cloud deployment to meet security best practices, as well as your business needs.

NAME

gcloud iam oauth-clients - create and manage OAuth clients

SYNOPSIS

`gcloud iam oauth-clients` `  GROUP  ` | `  COMMAND  ` \[ `  GCLOUD_WIDE_FLAG …  ` \]

DESCRIPTION

The gcloud iam oauth-clients group lets you create and manage OAuth clients for projects on the Google Cloud Platform.

GCLOUD WIDE FLAGS

These flags are available to all commands: `  --help  ` .

Run ` $ gcloud help  ` for details.

GROUPS

`  GROUP  ` is one of the following:

  - `  credentials  `  
    Create and manage OAuth client credentials.

COMMANDS

`  COMMAND  ` is one of the following:

  - `  create  `  
    Create an OAuth client.
  - `  delete  `  
    Delete an OAuth client.
  - `  describe  `  
    Describe an OAuth client.
  - `  list  `  
    List OAuth clients.
  - `  undelete  `  
    Undelete an OAuth client.
  - `  update  `  
    Update an OAuth client.

NOTES

This variant is also available:

    gcloud alpha iam oauth-clients

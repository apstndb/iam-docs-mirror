NAME

gcloud iam workforce-pools subjects - create and manage workforce pool subjects

SYNOPSIS

`gcloud iam workforce-pools subjects` `  GROUP  ` | `  COMMAND  ` \[ `  GCLOUD_WIDE_FLAG …  ` \]

DESCRIPTION

The gcloud iam workforce-pools subjects group lets you create and manage workforce pool subjects.

GCLOUD WIDE FLAGS

These flags are available to all commands: `  --help  ` .

Run ` $ gcloud help  ` for details.

GROUPS

`  GROUP  ` is one of the following:

  - `  operations  `  
    Manage IAM workforce pool subject long-running operations.

COMMANDS

`  COMMAND  ` is one of the following:

  - `  delete  `  
    Delete a workforce pool subject.
  - `  undelete  `  
    Undelete a workforce pool subject.
